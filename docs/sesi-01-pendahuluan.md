## Big Picture

- Mata kuliah ini tentang **perpaduan SI (Sistem Informasi) dan TI (Teknologi Informasi)**
- Tujuan: Memahami bagaimana teknologi digunakan untuk mengelola informasi
- Konteks: Untuk individu, organisasi, bisnis, dan masyarakat

---

## Konsep Kunci

---

### 1. Data vs Informasi

**Data:**
- Kumpulan simbol atau fakta mentah
- Belum punya makna
- Contoh: "25", "Jakarta", "2024"

**Informasi:**
- Data yang sudah **diproses**
- Punya **makna** dan **konteks**
- Contoh: "Suhu Jakarta hari ini 25°C"

---

### 2. SI vs TI (Wajib Hafal!)

**Sistem Informasi (SI):**
- Kombinasi **manusia + teknologi + proses**
- Fungsi: collect, record, process, store, retrieve, transmit, receive informasi
- **Fokus: PENGGUNAAN informasi**
- Untuk: pengambilan keputusan & operasi organisasi
- Contoh: Sistem inventory, sistem absensi, e-banking

**Teknologi Informasi (TI):**
- Alat/perangkat untuk mengelola informasi
- **Fokus: ALAT & PERANGKAT**
- Contoh: komputer, software, jaringan, database

> **Ingat:** SI = cara pakai | TI = alatnya

---

### 3. Tiga Dimensi Informasi (Time - Location - Form)

**1. Dimensi Waktu (Time)**
- Bisa diakses **saat dibutuhkan**
- Relevan dengan periode tertentu
- Perlu filter mana yang penting

**2. Dimensi Lokasi (Location)**
- Bisa diakses **dari mana saja**
- Saling terkait antar sumber

**3. Dimensi Bentuk (Form)**
- Bisa **digunakan & dipahami**
- **Akurat, valid, bebas kesalahan**

---

### 4. Lima Komponen TI (Hafal!)

**1. Hardware (Perangkat Keras)**
- Perangkat fisik internal/eksternal
- Contoh: CPU, RAM, hard drive, keyboard, monitor

**2. Software (Perangkat Lunak)**
- Program yang berjalan di hardware
- Interaktif dengan user
- Contoh: calculator, text editor, browser, game

**3. Firmware**
- Software untuk hardware
- **Tidak interaktif** dengan user
- Contoh: program di TV, router, printer

**4. Brainware**
- **SDM (Sumber Daya Manusia)** dalam TI
- Termasuk pengalaman, skill, pengetahuan
- Contoh: user, developer, stakeholder, admin

**5. Infoware**
- Dokumentasi data/informasi
- Contoh: user guide, manual, SOP, cyber law

---

### 5. Era Informasi (Information Age)

- Dimulai sejak **1950-an** (pertengahan abad ke-20)
- Masa Revolusi Industri
- Ciri: Lebih banyak orang bekerja di bidang **informasi** daripada pertanian/industri
- Kita hidup di era ini sekarang

---

### 6. Kenapa Belajar SI/TI? (Urgensi)

- **Necessary Skill** - Banyak dicari di pasar kerja
- **Increasing Efficiency** - Otomatisasi & pengolahan data lebih baik
- **Better Problem Solving** - Keputusan berbasis analisis data
- **Innovation** - Mendorong solusi baru
- **Adapting to Change** - Siap hadapi perubahan teknologi
- **Digital Independence** - Tidak gaptek, bisa mandiri

## Perbedaan yang Sering Membingungkan

### Software vs Firmware

| Aspek | Software | Firmware |
|-------|----------|----------|
| Interaksi | **Interaktif** dengan user | **Tidak interaktif** |
| Fungsi | Program untuk user gunakan | Program untuk jalankan hardware |
| Contoh | MS Word, Chrome, Game | Program di TV, router, printer |
| Update | Mudah di-update user | Jarang di-update, biasanya oleh teknisi |

### Data vs Informasi

| Aspek | Data | Informasi |
|-------|------|-----------|
| Sifat | Mentah, belum diproses | Sudah diproses |
| Makna | Belum punya makna | Punya makna & konteks |
| Contoh | "100", "Budi" | "Budi mendapat nilai 100" |
| Kegunaan | Belum bisa dipakai untuk keputusan | Bisa dipakai untuk keputusan |

### SI vs TI

| Aspek | Sistem Informasi (SI) | Teknologi Informasi (TI) |
|-------|----------------------|--------------------------|
| Fokus | **Penggunaan** informasi | **Alat** untuk proses informasi |
| Komponen | Manusia + Teknologi + Proses | Hardware, Software, dll |
| Pertanyaan | "Bagaimana informasi digunakan?" | "Apa alat yang digunakan?" |
| Contoh | Sistem absensi karyawan | Komputer, aplikasi absensi |

## Tips Menjawab Soal

### Cara Bedakan Soal SI vs TI

**Jika soal bertanya tentang:**
- Proses bisnis, pengambilan keputusan, organisasi → **SI**
- Alat, perangkat, teknologi spesifik → **TI**

**Keyword SI:**
- Pengambilan keputusan
- Proses bisnis
- Organisasi
- Manajemen informasi

**Keyword TI:**
- Hardware, software
- Perangkat
- Alat
- Teknologi

### Cara Bedakan Data vs Informasi

**Tanya diri sendiri:**
- Apakah sudah punya konteks? → Informasi
- Masih berupa angka/fakta mentah? → Data

### Cara Ingat 5 Komponen TI

**Akronim: "HSFBI"** (baca: Has-F-BI)
- **H**ardware
- **S**oftware
- **F**irmware
- **B**rainware
- **I**nfoware

Atau ingat urutan dari yang paling "fisik" ke paling "abstrak":
1. Hardware (fisik)
2. Software (program)
3. Firmware (program untuk hardware)
4. Brainware (manusia)
5. Infoware (dokumentasi)

## Contoh Soal Latihan

**Apa perbedaan utama antara SI dan TI?**

a) SI lebih mahal dari TI
b) SI fokus pada penggunaan informasi, TI fokus pada alat

a) 1900-an
b) 1950-an
c) 1980-an

### Soal 4: Essay
**Sebutkan dan jelaskan 5 komponen Teknologi Informasi!**

> **Jawaban:**
> 1. **Hardware** - Perangkat fisik seperti CPU, RAM, keyboard
> 2. **Software** - Program yang berjalan di hardware seperti text editor
> 3. **Firmware** - Software untuk hardware yang tidak interaktif seperti program di printer
> 4. **Brainware** - SDM dalam TI seperti user, developer
> 5. **Infoware** - Dokumentasi seperti user guide, SOP

---

### Soal 5: Essay
**Jelaskan 3 dimensi informasi!**

> **Jawaban:**
> 1. **Dimensi Waktu** - Informasi bisa diakses saat dibutuhkan dan relevan dengan periode tertentu
> 2. **Dimensi Lokasi** - Informasi bisa diakses dari mana saja dan saling terkait
> 3. **Dimensi Bentuk** - Informasi bisa dipahami, digunakan, akurat, dan bebas kesalahan

---

### Soal 6: Analisis
**"25, Jakarta, Senin" - Apakah ini data atau informasi? Jelaskan!**

> **Jawaban:** Ini adalah **DATA** karena masih berupa fakta mentah yang belum diproses dan belum memiliki konteks/makna yang jelas. Jika diubah menjadi "Suhu Jakarta pada hari Senin adalah 25°C", maka baru menjadi **INFORMASI** karena sudah punya makna dan konteks.

## Quick Recall (Hafalan Cepat)

- **Data** = fakta mentah
- **Informasi** = data + makna + konteks
- **SI** = cara pakai informasi (manusia + proses)
- **TI** = alat untuk proses informasi
- **5 Komponen TI** = Hardware, Software, Firmware, Brainware, Infoware
- **3 Dimensi Informasi** = Waktu, Lokasi, Bentuk
- **Era Informasi** = mulai 1950-an
- **Firmware** = software untuk hardware, TIDAK interaktif
